# Job Tracker Full-stack Application

A modern job application tracking system built with TypeScript, Node.js, Express, PostgresSQL and Vue.js

## Tech Stack

 - Backend: Node.js + TypeScript + Express + Prisma + PostgresSQL
 - Frontend: Vue 3 + TypeScript + Tailwind CSS
 - Shared: TypeScript types

## Getting started

See individual package READMEs for setup instructions

## Development
```bash
# Install dependencies
npm run install:all

# Start development servers
npm run dev
```
